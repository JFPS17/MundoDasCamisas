# Mundo das Camisas
Este é um projeto de loja de camisetas esportivas feito por alunos do curso de Análise e Desenvolvimento de Sistemas na UNISUAM. A loja é meramente fictícia e foi feita em HTML, CSS e JS com o framework Bootstrap, estando hospedada no Netlify.

# Objetivo
Temos como objetivo entregar aos usuários um serviço acessível, responsivo e versátil, demonstrando competência para competir com outros serviços em meio à alta do tópico de futebol.
